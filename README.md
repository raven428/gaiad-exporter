# gaiad-exporter
gaiad prometheus exporter

```
# HELP gaiad_latest_block_time Latest block time in gaiad
# TYPE gaiad_latest_block_time gauge
gaiad_latest_block_time 1.615724775e+09
# HELP gaiad_number_of_blocks Total number of blocks in gaiad
# TYPE gaiad_number_of_blocks gauge
gaiad_number_of_blocks 289433
# HELP gaiad_number_of_peers Current number of peers in gaiad
# TYPE gaiad_number_of_peers gauge
gaiad_number_of_peers 10
```
